## About the project

This project is a web application that displays tech events in Yamanashi prefecture.

This application is available at the following URL.

https://hub.yamanashi.dev

## Getting Started

### Prerequisites

- Node.js 21.4.0
- Yarn 1.22.21

### Installation

1. Clone the repo
   ```sh
   $ git clone
    ```
2. Install packages
    ```sh
    $ yarn install
    ```
3. Run the app
    ```sh
    $ yarn dev
    ```
4. Open http://localhost:5173
